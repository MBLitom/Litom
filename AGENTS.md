# MB Litom Agent Rules

- MB Litom is a serious B2B software studio, not a generic freelancer portfolio.
- Keep the architecture as a Django monolith unless a human explicitly approves a different direction.
- Use proposal, milestone, deposit, payment request, project tracking, and delivery language.
- Do not build a generic e-commerce cart.
- Paysera Checkout is planned later; do not add Stripe or premature payment integration.
- Production-impacting decisions require human approval.
- Keep changes clean, small, tested, readable, and maintainable.
- Avoid hype, fake enterprise claims, and unnecessary dependencies.
